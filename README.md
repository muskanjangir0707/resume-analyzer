# SmartHireX – AI-Driven Resume Screening & Career Intelligence System

## Overview

SmartHireX is an AI-powered resume screening platform designed to help job seekers analyze their resumes, improve ATS (Applicant Tracking System) compatibility, identify skill gaps, and receive personalized career recommendations.

The platform automates resume evaluation and provides actionable insights to enhance employability and improve job application success rates.

---

## Features

### Resume Analysis

* Upload resumes in PDF format
* Automatic text extraction and processing
* Resume content evaluation

### ATS Score Calculation

* Calculates ATS compatibility score
* Identifies strengths and weaknesses
* Provides resume improvement suggestions

### Skill Gap Analysis

* Detects existing technical skills
* Identifies missing skills for target roles
* Suggests skills to learn for career growth

### Job Role Matching

* Matches resumes with suitable job roles
* Recommends career paths based on skill sets

### User Authentication

* Secure user registration and login
* Personalized user dashboard

### Resume History Tracking

* Stores previous resume analyses
* Allows users to review past reports

### Report Generation

* PDF report generation
* Excel report export
* Downloadable analysis results

### Admin Dashboard

* Analytics and user insights
* Resume screening statistics
* Visual charts and reports

## Project Architecture

1. User uploads a resume.
2. The system extracts text from the PDF.
3. Skills and keywords are identified.
4. ATS score is calculated.
5. Missing skills are detected.
6. Career recommendations are generated.
7. Results are stored in the database.
8. Reports can be downloaded in PDF or Excel format.
